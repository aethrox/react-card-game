### Getting Started:

1. Install dependencies:  
   ```bash
   yarn install
   ```
2. Start the development server:  
   ```bash
   yarn start
   ```

Open [http://127.0.0.1:3000](http://127.0.0.1:3000) in your browser to view the game.
